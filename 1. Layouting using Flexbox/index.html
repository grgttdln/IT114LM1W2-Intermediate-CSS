Todo 1.1: Flex direction, wrap, and gap

Observation:
The flex-direction property determines the main axis alignment of flex items.
The flex-wrap property determines whether the flex container should wrap its content onto a new line if there is not enough space along the main axis.
The gap property sets the spacing between flex items within a container.

Todo 1.2: Justify content and align items

Observation:
The justify-content property aligns flex container items along the main axis (horizontally).
The align-items property aligns flex container items along the cross axis (vertically).

Todo 1.3: align self and grow

Observation:
The align-self property allows individual flex items to override the align-items property and specify their own alignment.
The flex-grow property defines the ability of a flex item to grow within a flex container along the main axis, distributing available space proportionally to items with a positive value.

Todo 1.4: Auto Margins and order

Observation:
The use of auto margins property is used for centering block-level elements within their containing block.
The order property is used within a flex container to define the visual order of flex items, with lower order specified values appearing first.
